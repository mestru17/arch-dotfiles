# arch-dotfiles
Dot files for my Arch Linux setup.

## Installation
First install the dependencies:
```shell
sudo pacman -S alacritty nitrogen fish archlinux-wallpaper exa bat lazygit
yay -S nerd-fonts-mononoki pfetch
```

Then run the install script:
```shell
curl -L https://raw.githubusercontent.com/mestru17/arch-dotfiles/master/install_dotfish.fish | fish
```
